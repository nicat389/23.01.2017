Machine language
Machine language aşağı səviyyəli proqramlaşdırma dillərindən biridir. Machine language yalnız kompüter tərəfindən başa 
düşülən proqramlaşdırma dilidir. Bu dildə kodlar yalnız 0 və 1-lərlə ifadə olunur. Buna görə də bu dili başa düşmək
çox çətindir.

Assembler 
Assembler aşağı səviyyəli proqramlaşdırma dillərindən biridir. Bu proqramlaşdırma dili 1952-ci ildə amerikalı qadın Qreys
Hopper tərəfindən hazırlanan mnemonik dildir. Kodlar yığımını həyata keçirən proqram kompilyator adlanır. Bunu Qreys Hopper
düşünüb tapmışdır. Assembler dili digər proqramlaşdırma dillərinə bənzəmir və tamamilə fərqli bir dildir. Assemblersiz 
müasir proqramlaşdırmanı təsəvvür etmək qeyri-mümkündür. Hal-hazırda bu dil inkişaf etdirilir. Assembler dilinin imkanları 
sonsuzdur.

Cobol(Common Business Oriented Language)
Cobol 1950-ci illərin sonu 60-cı illərin əvvəllərində hazırlanmışdır. Cobol proqramlaşdırma dili Fortran proqramlaşdırma
dilindən sonra 2-ci ən yaşlı yüksəl səviyyəli proqramlaşdırma dilidir. Ticarət və xüsusilə biznes idarəsi ilə əlaqəli 
movzularda bütün dünya istifadə edilmək üçün hazırlanıb. Cobol 4 hissədən ibarətdir:
1)Təyin etmə hissəsi	2)Mühit hissəsi	3)Prosedur hissəsi	4)Kəsim hissəsi
Hal-hazırda Cobol çox əhəmiyyətli mövqeyə malikdir. Bunun səbəblərdən biri Microsoft tərəfindən 2001-ci ildə təqdim olunmuş
.NET Framework 1.1 versiyası ilə dəstəklənməyə başlanmasıdır.

C dili
Bu dil 1972-ci ildə Dennis Ritchie ve Ken Thompson tərəfindən Bell laboratoriyalarında hazırlanmış və Unix əməliyyat 
sistemini inkişaf etdirmək məqsədilə "B" dilindən törəmiş bir dildir. Bu gün istifadə etdiyimiz əməliyyat sistemlərinin 
95%-də C-dən istifadə olunur. C general-purpose proqramlaşdırma dilidir və bütün platformalarda bu dildən istifadə 
mümkündür. C struktur yönümlü proqramlaşdırma dilidir. C++, Java, C# və digər dillər bu proqramlaşdırma dilinin əsasında
yaradılmışdır. Bjarne Stroustrup C dilini inkişaf etdirərək obyekt yönümlü C++ dilini yaratmışdır və C dilinin sonuna "++"
əlavə etmişdir. 

Ruby
Obyektyönümlü, dinami, reflektiv proqramlaşdırma dili olan Ruby yaponiyalı Yukihiro Matsumoto tərəfindən hazırlanmışdır.
Sintaksis cəhətdən Ada, Perl, Smalltalk, Lisp, Eiffel kimi proqramlaşdırma dillərindən hazırlanan Rubynin Python ilə bir 
çox ortaq sozləri(sintaksis sözlər) mövcuddur. Bu dildə 4 fərqli tipdə dəyişənlər təqdim oluna bilər: 
Global,class,instance və local

Javascript
Əsasən web brauzerlərdə istifadə olunan bu Javascript dinamik bir proqramlaşdırma dilidir. Sintaksisi C dilinin sintaksisinə
əsaslanır. Javascript Html və CSS ilə birlikdə İnternet saytların hazırlanması üçün 3 əsas nüvədən biridir. Obyekt yönümlü 
proqramlaşdırma dillərini dəstəkləyir. Javascript saytları interaktiv etmək üçün istifadə olunan pr. dilidir.
